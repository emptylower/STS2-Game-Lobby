# STS2 LAN Connect v0.5.2-rc.1 预览发布说明

> 这是客户端 `0.5.2` 的 GitHub Pre-release，标签为 `v0.5.2-rc.1`。源码来自未合并 main 的 `feat/rich-chat-reference-ux-0.5.2` 分支；稳定版仍为 `v0.5.1`，本次不更新 Steam 创意工坊。

v0.5.2 是客户端交互与渲染升级，继续使用 lobby-service 0.5.1 和 v0.5.1 聊天协议；服主无需升级服务端。新版客户端与 v0.5.1 服务端、v0.5.1 客户端可安全互通。

## 一次性引用模式

- Android：点击聊天输入区旁的“引用”按钮进入一次性引用模式，再点击卡牌、遗物、药水、状态或玩家。成功引用后自动退出，并回到真实文本输入位置继续输入。
- 桌面：按 `Alt+R` 进入或取消一次性引用模式；原有 `Alt+左键` 直接引用继续保留，并复用同一捕获规则。
- 再次点击引用按钮、`Esc`、切频道、关闭聊天、离开房间或房间 generation 变化都会取消引用模式。
- 点击不支持的区域不会吞掉原本的游戏操作，引用模式保持 armed 并显示短提示；成功捕获会消费该次点击，避免同时出牌或触发物品。
- 服务器频道只允许卡牌、遗物和药水引用；战斗状态与玩家引用只允许当前房间频道。怪物目标引用仍未开放。

## 行内消息与原生预览

- 一条消息现在由单一原生富文本控件渲染，文字、Emoji 和引用会自然换行，不再出现独立块间距和错位。
- 卡牌使用游戏原生 card hover tip；遗物、药水和 Power 使用原生 hover tip、`MegaLabel` 与 `MegaRichTextLabel`。
- Power 说明会按实际层数、持有者、施加者、目标、玩家数、能量前缀与动态变量生成。例如“触媒 4”会显示与当前游戏状态一致的最终效果。
- Android 点击消息中的引用可打开固定说明面板；点击外部、`Esc` 或关闭按钮可关闭。桌面继续支持悬停，也可以点击固定预览。
- 引用只携带既有结构化数据与本地 opaque meta，不会向聊天对端泄漏内部 model ID，也不会执行游戏动作或下载远端预览资源。

## 输入与可访问性

- 引用实体边界使用真实 `TextEdit` 合成插入槽，成功引用后焦点回到文本输入位置，可继续中文组合输入、英文直键、粘贴、删除和移动光标。
- 引用和关闭按钮的最小触摸目标为 `44x44`，固定预览提供中英文可访问名称。
- Android 模拟器验收使用触屏 UI 与 Mac 实体键盘输入；引用后拼音组合输入、发送和点击消息预览均已验证。

## 兼容范围

- 客户端版本：`0.5.2`。
- 服务端版本：继续使用 `0.5.1`，不要求发布新服务端包。
- 游戏加载目标：`0.107.1` 与 `0.109.0`；`0.108.0` 不在本次发布适配范围内。
- 不改变聊天 capability intersection、限额、generation、legacy fallback、MOD 同步或加入流程。

## 安装与回滚

1. 完整退出游戏。
2. 用 `v0.5.2-rc.1` 预览包中的安装器安装完整 `sts2_lan_connect` 目录，不要混用旧 DLL、PCK 或 manifest。
3. 重启游戏，在主菜单确认只加载一个 STS2 LAN Connect。
4. 如需回滚，完整卸载 v0.5.2，再安装正式 v0.5.1 Release；不要覆盖或移动历史 tag/资产。

预览版不改变正式发布顺序：后续仍需先合并远程 main，再发布无测试尾缀的 v0.5.2 GitHub Release，最后更新 Steam Workshop。
